Overview

An automated graph generation system built to visualize performance metrics from database experiments. It integrates SQLite for data handling and Matplotlib for customizable and print-friendly plots based on configuration files.

ðŸ”§ Features

  • Reads configurations from a JSON file
  • Generates line and bar charts from SQLite database
  • Supports black-and-white and colorblind-friendly styling
  • Aligns x-axis with bars precisely for comparison
  • Minimalistic design (no grid lines, only x and y axis)

ðŸ“‚ Project Structure

  • graph_generator.py: Python script for generating graphs
  • default_graphs.json: Configuration for plot types, axis labels
  • experiment_results.db: SQLite database with performance data
  • experiment_test.cpp: C++ file generating test data

ðŸ§  Example Code Snippet

import sqlite3
import matplotlib.pyplot as plt

conn = sqlite3.connect('experiment_results.db')
cursor = conn.cursor()
cursor.execute("SELECT exp_conf, throughput FROM experiment_results")
data = cursor.fetchall()

x, y = zip(*data)
plt.bar(x, y, color='gray')
plt.xlabel('Configuration')
plt.ylabel('Throughput')
plt.axhline(0, color='black')
plt.show()
